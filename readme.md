# YouTube Product Analytics: Funnel, Retention & User Engagement Analysis

## Project Overview

This project analyzes user behavior on a YouTube-style platform using PostgreSQL and Power BI.

The objective was to understand:

- User engagement
- Funnel conversion
- Retention
- User segmentation
- Product growth opportunities

Dataset contains:

- 675,020 user events
- 474,000 unique users
- 30 days of activity

---

## Tech Stack

- PostgreSQL
- SQL
- Power BI
- Python
- Pandas

---

## Key Metrics

| Metric | Value |
|----------|----------|
| MAU | 474,000 |
| Average DAU | 19,735 |
| Stickiness | 4.16% |
| D1 Retention | 17,957 |
| D7 Retention | 6,196 |
| View → Like | 6.35% |
| Like → Subscribe | 31.00% |
| Overall Conversion | 1.97% |

---

## Funnel Analysis

Video View → Like Video → Subscribe

Key Finding:

Only 6.35% of viewers engage with content, but engaged users convert strongly to subscribers (31%).

---

## User Segmentation

- Casual Users: 356,170
- Regular Users: 108,919
- Power Users: 8,911

---

## Business Recommendations

1. Improve engagement through personalized recommendations.
2. Increase D7 retention with onboarding nudges.
3. Create loyalty features for power users.
